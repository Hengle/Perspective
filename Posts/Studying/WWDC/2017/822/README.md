> WWDC 2017 是苹果开发者大会的 2017 版。Session 822 简要介绍了设计 App 的图标（Icon）时需要注意的点。

## 图标设计

- 象征（Metaphor）
  - 即人们可以根据图标所联想的事物。
- 简约（Simplicity）
  - 清晰简约，可读性强。
- 连接（Connection）
  - 图标是用户和应用的桥梁，要有趣，有启发性。
- 家系（Lineage）
  - 图标的改变要慎重，有一致性。

## 设计过程

- 唯一（Unique）
  - 在应用的分类中脱颖而出。
- 体验（Experiment）
  - 开始可以手绘，也可以与他人共同体验。
- 测试（Test）
  - 在桌面，文件夹，设置中测试图标的效果。
- 耐心（Patience）
